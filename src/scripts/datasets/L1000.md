# l1000_drug/

Drug perturbation (gene sets) -> regulatory elements

Perturbation gene sets based on characteristic direction (CD) method  
[https://maayanlab.cloud/sigcom-lincs/#/Download](https://maayanlab.cloud/sigcom-lincs/#/Download)

**/!\\ Requires `huniq`. Use `sort -u` as an alternative.**

```bash
#! /bin/bash

out="data/l1000_drug"
mkdir -p $out

# get the core dataset
#   compounds, perturbation up/down gene sets
wget -P $out https://lincs-dcic.s3.amazonaws.com/LINCS-sigs-2021/gmt/l1000_cp.gmt

# the unique set of gene names, 15s
cut $out/l1000_cp.gmt -f 2- | tr '\t' '\n' | huniq | egrep -v '^\s*$' > $out/gene_names.txt

# optional: drug names
cut $out/l1000_cp.gmt -f 1 | cut -f 1 -d ' ' | huniq > $out/drug_names.txt

# clean gene names
# wget -P $out storage.googleapis.com/public-download-files/hgnc/json/json/hgnc_complete_set.json
py src/scripts/gene_clean.py -i $out/gene_names.txt -o $out/gene_names_clean.txt

# map gene names to loci
cat $out/gene_names_clean.txt | py src/scripts/datasets/genes2regions.py -o $out/genes.bed

# map gene sets to bed files, 3m & ~12G mem
mkdir $out/drug_regions
cat $out/l1000_cp.gmt | py src/scripts/datasets/gmt2beds.py -f $out/gene_names.txt -t $out/gene_names_clean.txt -r $out/genes.bed -o $out/drug_regions

# next, we merge `{name}_up.bed` and `{name}_down.bed` files in `$out/drug_regions/` into the form `{name}.bed`
py src/scripts/datasets/updown_bed_merge.py -i $out/drug_regions
```
